## ‚ö° Speed Control

### Why Use Speed Control?

**Learn Faster:** Review material at 1.5x-2.0x  
**Learn Deeper:** Study difficult concepts at 0.5x-0.75x  
**Save Time:** Cover more content in less time  

### Speed Controls
**Location:** Bottom left of player

- **‚àí Button:** Decrease speed by 0.25x
- **Speed Display:** Shows current speed (e.g., "1.0x")
- **+ Button:** Increase speed by 0.25x

### Speed Reference Guide

| Speed | Best For | Example Use Case |
|-------|----------|------------------|
| **0.5x** | Very difficult material | Advanced calculus lectures |
| **0.75x** | New concepts | First-time learning |
| **1.0x** | Normal listening | Standard podcasts |
| **1.25x** | Efficient learning | Familiar topics |
| **1.5x** | Quick review | Exam preparation |
| **1.75x** | Fast overview | Browsing content |
| **2.0x** | Speed review | Re-listening material |

### Speed Tips
- **Start at 1.0x** for new content
- **Gradually increase** as you get comfortable
- **Slow down** for complex sections
- **Speed up** for reviews and summaries
- **Pitch stays natural** at all speeds

## üîç Troubleshooting

### Issue: Audio won't play

**Solutions:**
1. Check browser permissions for audio
2. Verify file was generated successfully
3. Try different format (MP3 vs Opus)
4. Refresh page and try again
5. Check browser console for errors

---

### Issue: Speed control not working

**Solutions:**
1. Ensure audio is playing
2. Try different browser (Chrome recommended)
3. Check if browser supports playback rate
4. Restart browser and try again

---

### Issue: Progress bar not moving

**Solutions:**
1. Audio may still be loading
2. Check network connection
3. Try smaller format (Opus)
4. Verify file integrity with download

---

### Issue: Volume too low

**Solutions:**
1. Check player volume slider (right side)
2. Check system volume
3. Check browser volume
4. Try different audio output device

---

### Issue: Playlist not showing

**Solutions:**
1. Refresh page
2. Verify podcasts were generated
3. Check browser console for errors
4. Try generating new podcast
